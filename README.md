## 📌 Projetos em Destaque

| Projeto | Descrição | Stack |
|---|---|---|
| [🔗 go-native-selenium](https://github.com/champiao/go-native-selenium) | Implementação nativa do protocolo **W3C WebDriver** em Go puro. Sem dependência do servidor Selenium Java. Suporta waits explícitos, cookies, screenshots e detecção automática do ChromeDriver | Go |
| [📧 GomailAPI](https://github.com/champiao/GomailAPI) | API REST para disparo de e-mails com **múltiplas configurações SMTP** por variável de ambiente — ideal para centralizar envios de vários sistemas em uma só API | Go |
| [🤖 domainBot](https://github.com/champiao/domainBot) | Bot para **Telegram** que monitora e informa a data de expiração de domínios configurados via variáveis de ambiente | Python |
